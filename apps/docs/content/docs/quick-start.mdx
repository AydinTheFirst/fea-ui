## Requirements

- React 19+
- Tailwind CSS v4 (if using `@fea-ui/styles` for default styling)

## Quick Install

Install FeaUI and its peer dependencies via npm or yarn:

```package-install
@fea-ui/react
```

## Import styles

Styling is optional. You can style FeaUI components yourself, or use the default styles provided.

```package-install
@fea-ui/styles
```

In your main CSS file, import Tailwind and FeaUI styles:

```css
@import "tailwindcss";
@import "@fea-ui/styles"; /* FeaUI styles */
```

## Use Components

```jsx
import { Button } from "@fea-ui/react";
function App() {
  return <Button variant="primary">Get Started</Button>;
}
```

## Requirements

- React 19+
- Tailwind CSS v4

## Quick Install

Install FeaUI and its peer dependencies via npm or yarn:

```package-install
@fea-ui/react
```

## Import styles

```css
@import "tailwindcss";
@import "@fea-ui/react"; /* FeaUI styles */
```

## Use Components

```jsx
import { Button } from "@fea-ui/react";
function App() {
  return <Button variant="primary">Get Started</Button>;
}
```
